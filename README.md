# Task Management Application

A task management application built with Vanilla JavaScript that helps users organize daily activities efficiently.

## Features
- Create tasks
- Edit existing tasks
- Delete tasks
- Search tasks
- Sort tasks
- Filter tasks
- Mark tasks as completed

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)
